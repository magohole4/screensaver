# Magohole4 ScreenSaver
Simple screen saver chrome app written in javascript.

:bangbang: Only tested in one chromebook.

## Controls

| Function                                          | Control            |
|---------------------------------------------------|--------------------|
|Draws the background particles with random colors. |f                   |
|Draws the logo particles with random colors.       |r                   |
|Acid mode: Doesn't clear the background.           |a                   |
|Menu: change the colors, disable particles...      |i (Exit with Ctrl+i)|
|Exit the program                                   |Esc                 |
